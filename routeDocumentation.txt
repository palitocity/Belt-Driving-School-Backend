## ðŸ”¹ 2. Reset Password

**Endpoint:**

```
POST https://belt-driving-school-backend-3.onrender.com/api/auth/reset-password
```

**Description:**
Resets the password of a verified user or admin using a valid **reset token**.
Once successful, the token is invalidated to prevent reuse.

**Request Body:**

```json

  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "newPassword": "NewStrongPassword123!",
  "role": "user"
}
```

| Field       | Type   | Required | Description                                       |
| ----------- | ------ | -------- | ------------------------------------------------- |
| token       | string | âœ… Yes    | JWT reset token generated from `/forgot-password` |
| newPassword | string | âœ… Yes    | The new password to set                           |
| role        | string | âœ… Yes    | Either `"user"` or `"admin"`                      |

**Successful Response (200 OK):**

```json
{
  "message": "Password reset successful"
}
```

**Error Responses:**

| Code | Message                              |
| ---- | ------------------------------------ |
| 400  | `Invalid or expired token`           |
| 400  | `Token, password, and role required` |
